# 🌸 Classification des Iris - Machine Learning

## 📋 Description
Projet de classification des iris utilisant 8 algorithmes d'arbres de décision.

## 🌳 Algorithmes Implémentés
1. Arbre de décision simple (CART)
2. Arbre de décision élagué
3. Arbre de décision (Entropie)
4. Random Forest
5. Gradient Boosting
6. AdaBoost
7. Extra Trees
8. Bagging

## 🚀 Installation
pip install -r requirements.txt

## 💻 Utilisation
python main.py
